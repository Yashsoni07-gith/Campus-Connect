# Campus Connect – Smart Campus Platform

## Problem Statement
In college campuses, important information such as announcements, events, resources, and student interactions are scattered across multiple platforms like WhatsApp groups, notice boards, and emails. This results in missed updates, confusion, and inefficient communication among students.

## Solution
Campus Connect is a centralized campus platform built using Google technologies that brings essential campus services into one place. It provides real-time announcements, a student-to-student marketplace, shared resources, and an AI-powered chatbot to resolve campus-related queries. The platform aims to simplify communication, save time, and improve collaboration within the campus ecosystem.

## Features
- Centralized campus announcements and updates  
- Student-to-student marketplace for buying and selling items  
- AI-powered chatbot for instant query resolution  
- Google Calendar integration for events and reminders  
- Google Drive-based access to booklets and resources  
- Secure Gmail authentication  
- Simple, responsive, and user-friendly interface  

## Google Technologies Used
- Firebase – Authentication, database, and hosting  
- Gemini API – AI chatbot and helpdesk support  
- Google Drive – Resource storage and sharing  
- Google Calendar – Event scheduling and reminders  
- Gmail Authentication – Secure user sign-in  

## Process Flow
1. User signs in using Gmail authentication  
2. User accesses the dashboard  
3. Announcements and events are displayed  
4. Events can be added to Google Calendar  
5. Students interact through the marketplace  
6. AI chatbot handles campus-related queries  
7. Resources are accessed through Google Drive  

## Future Scope
- Role-based access for faculty and administrators  
- Analytics dashboard for engagement tracking  
- Push notifications for urgent announcements  
- Integration with Google Forms for registrations and feedback
